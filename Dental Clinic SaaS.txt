Dental Clinic SaaS — Development Progress Documentation
Project Overview

A multi-tenant SaaS backend system designed to allow multiple dental clinics to manage their clinic operations through a single platform.

Each clinic has isolated data while sharing the same software infrastructure.

Core capabilities include:

clinic authentication

patient management

secure API access

multi-tenant database architecture

Authentication System
Signup API

Endpoint:

POST /api/auth/signup

Function:

registers new clinic

hashes password using bcrypt

stores clinic data in database

Process:

password
 ↓
bcrypt.hash()
 ↓
store hashed password
Login API

Endpoint:

POST /api/auth/login

Function:

verifies password using bcrypt

generates JWT token

Authentication flow:

user login
 ↓
bcrypt.compare()
 ↓
password verified
 ↓
JWT token generated

Example response:

{
 "message": "Login successful",
 "token": "JWT_TOKEN"
}
JWT Authentication

JWT payload structure:

{
 clinic_id: 1
}

Token expiration:

1 hour

Token usage in requests:

Authorization: Bearer JWT_TOKEN
Protected Middleware

File:

middleware/authMiddleware.js

Purpose:

verifies JWT tokens

extracts clinic_id

protects private routes

Flow:

Request
 ↓
Authorization header
 ↓
JWT verification
 ↓
clinic_id attached to request
Patient Management API
Create Patient

Endpoint:

POST /api/patients

Protected route.

JWT provides:

clinic_id

Example body:

{
 "name": "Arun Kumar",
 "phone": "9876543210",
 "age": 35,
 "address": "Bangalore"
}

Result:

Patient stored with clinic_id.

Get Patients

Endpoint:

GET /api/patients

Query logic:

SELECT * FROM patients
WHERE clinic_id = logged_in_clinic

Example response:

[
 {
  id:1,
  clinic_id:1,
  name:"Arun Kumar",
  phone:"9876543210"
 }
]

This ensures:

Clinic A → sees only its patients
Clinic B → sees only its patients
Multi-Tenant SaaS Logic

The system isolates clinic data using:

clinic_id

Data separation model:

Clinic A
   └── Patients A

Clinic B
   └── Patients B

This allows one platform to support many clinics simultaneously.

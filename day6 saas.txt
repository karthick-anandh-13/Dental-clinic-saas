8. JWT Authentication Flow

Frontend authentication flow was implemented.

Flow:

Login Page
   ↓
POST /api/v1/users/login
   ↓
Backend verifies credentials
   ↓
JWT token returned
   ↓
Token stored in localStorage
   ↓
Redirect to /dashboard

Token storage:

localStorage.setItem("token", token)
9. Protected Routes

ProtectedRoute component was implemented to secure dashboard pages.

If the user is not authenticated:

Redirect → /login

If token exists:

Allow access → dashboard

This prevents unauthorized access to system modules.
